# API2PDF
# What is API2PDF ?


It is an API that simplifies pdf generation for Developers.


What does it does ?


•REST API for instant pdf document generation.


•Removes infrastructure complexity for PDF rendering engines.


•Saves teams from maintaining fragile headless browser setups.


•Clear pain point for Saas apps: invoices,reports and contracts
